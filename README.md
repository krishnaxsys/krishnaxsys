### 💻 Current Work & Private Builds

I am actively building high-performance applications. Most of my core enterprise-grade work is kept in **private repositories**, including an enterprise-grade food ordering platform (Swiggy/Zomato level) built entirely from scratch. *(System architecture available on request).*

#### 🗓️ [ZynMeet](https://www.zynmeet.com) — Scheduling Platform
A high-performance booking and scheduling SaaS built for scale and reliability.
- **Backend:** Concurrency-safe Golang & Node.js hybrid.
- **Features:** Real-time availability engine, Google Calendar integration (auto event + meet link generation).
- **Architecture:** Multi-region ready, Redis locking for race condition handling, fault-tolerant system design.

#### 🦀 Zynfile
A file manager focused on extreme performance, built entirely in **Rust**.
- Outperforms File Explorer, Filepilot, and similar tools in copy/paste and core operations.
- Focused heavily on low-level system optimization.
